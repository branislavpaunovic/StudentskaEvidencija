**Uputstvo za instalaciju i pokretanje projekta 'Studentska Evidencija' sa GitHub-a**

---

### 1. PREDUSLOVI ZA RAD

Pre nego što pokrenete projekat, obavezno je da vaša lokalna mašina ima sledeće:

* **Windows 10/11**
* **.NET Framework 4.7.2 ili noviji** (automatski instaliran sa Visual Studio-om)
* **Visual Studio 2022 Community Edition** (preporučeno)
* **SQL Server 2019/2022 Express** (ili noviji)
* **SQL Server Management Studio 18 ili 19 (SSMS 21)**
* **GIT alat (ili GitHub Desktop)**

---

### 2. KORAK PO KORAK INSTALACIJA

#### 2.1. Kloniranje GitHub Repozitorijuma

1. Otvori Command Prompt ili Git Bash
2. Pokreni komandu:

   ```bash
   git clone https://github.com/branislavpaunovic/StudentskaEvidencija.git
   ```
3. Uđi u folder:

   ```bash
   cd StudentskaEvidencija
   ```

#### 2.2. Otvaranje projekta u Visual Studio-u

1. Pokreni **Visual Studio 2022**
2. Klikni na **Open a project or solution**
3. Navigiraj do foldera `StudentskaEvidencija` i otvori `StudentskaEvidencija.sln`

#### 2.3. Instalacija baze podataka

1. Otvori **SQL Server Management Studio (SSMS)**
2. Uloguj se na `localhost` instancu (ili kako se zove tvoja SQL instanca)
3. Kreiraj prazan folder `C:\Temp` ako ne postoji
4. Učitaj backup baze:

   * U SSMS-u klikni desni klik na **Databases > Restore Database...**
   * Izaberi opciju **Device** > klikni **...** > dodaj fajl:
     `StudentskaEvidencija\sql\StudentskaEvidencija.bak`
   * Dodeli naziv bazi `StudentskaEvidencija`
   * Klikni OK

---

### 3. KONEKCIONI STRING – PAŽNJA!

#### 3.1. Lokacija konekcionog stringa:

* `App.config`
* `DAL\Konekcija.cs`
* `frm_Login.cs`
* `frm_Main.cs`
* `frm_Studenti.cs`

> **OBAVEZNO!** Proverite sve klase i forme koje otvaraju konekciju ka bazi – konekcioni string mora da bude tačan.

#### 3.2. Format konekcionog stringa (Windows Authentication):

```xml
<connectionStrings>
  <add name="connString"
       connectionString="Data Source=localhost;Initial Catalog=StudentskaEvidencija;Integrated Security=True"
       providerName="System.Data.SqlClient" />
</connectionStrings>
```

#### 3.3. Ako koristite SQL Authentication:

```xml
<connectionStrings>
  <add name="connString"
       connectionString="Data Source=localhost;Initial Catalog=StudentskaEvidencija;User ID=administrator;Password=password"
       providerName="System.Data.SqlClient" />
</connectionStrings>
```

---

### 4. POKRETANJE PROJEKTA

1. U Visual Studio-u klikni na **Start (zeleni taster)** ili pritisni `F5`
2. Aplikacija će se otvoriti, a početna forma je **Login** forma.
3. Korisničko ime: `administrator` | Lozinka: `password`

---

### 5. DODATNE NAPOMENE

* Proverite da li SQL Server servis radi (Services > SQL Server)
* Ako dođe do greške sa konekcijom, proverite `connString` u svim klasama i formama
* Ako koristite drugu instancu SQL Servera, zamenite `localhost` sa imenom instance, npr. `localhost\SQLEXPRESS`

---

### 6. STRUKTURA PROJEKTA

* `DAL` – Data Access Layer (klase za rad sa bazom)
* `Models` – klase modela (Student, Pol, Smer...)
* `Forms` – forme aplikacije
* `sql` – SQL skripte i backup baze (`.bak` fajl)
